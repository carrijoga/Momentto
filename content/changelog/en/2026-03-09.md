# v1.6.1 - Onboarding Polish & Admin Dashboard

Small but meaningful improvements to the onboarding experience and a new internal admin panel.

## Improvements

- **Onboarding redesign** — steps now use animated text effects (BlurText, ScrollFloat) and a SpotlightCard layout for a more polished first-run experience
- **Smoother animations** — updated progress bar styling and icon presentation across the onboarding flow
- **Admin dashboard** — internal panel for monitoring signups, countdowns, and key metrics (not user-facing)
- Middleware updated to exclude admin routes from locale prefixing
